Mutual Authentication Chat App

The chat app utilizes TCP/IP together with Diffie-Hellman mutual authentication protocol and md5 HMAC for message integrity

Diffie-Hellman was also chosen for establishing session key for perfect forward secrecy

The integrity-protection in the app utilizes a md5 hash which hashes our message using the pre-established shared secret. The receiver checks the decrypted message against the HMAC to detect alteration on the message
